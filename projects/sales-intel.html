Project 03 / Multi-agent AI system

Sales Intel

A 3-agent AI system that fuses live web research, LLM synthesis, and internal sales data into one strategic account-intelligence dashboard.

Working app · dual deploy modes
Streamlit Snowflake Cortex Claude Sonnet 4.5 Auth0

01 Problem

Sales teams need a fast read on a prospect or account — market position, competitive posture, sustainability profile — without manually stitching together five different tools before every important call. That prep work was slow, inconsistent between reps, and rarely grounded in the company's own historical sales data.

02 Solution

Three cooperating agents run inside a dashboard with progressive result rendering. Agent 1 (Contextualizer) pulls internal analysis from Snowflake Cortex. Agent 2 (Researcher) layers in real-time web research. Agent 3 (Strategist) synthesizes both into a sales strategy, which can be exported to PDF or Markdown for use outside the tool.

03 Architecture

The Contextualizer and Strategist both run on Snowflake Cortex; the Researcher adds a live web-research API on top of Cortex. An orchestrator sequences the three agents and streams results into a Streamlit dashboard as each stage completes, then hands the final synthesis to a PDF/Markdown export step.

Fig. 1 — Three agents feed a typed orchestrator; results stream progressively into the dashboard, then export.

04 Tech stack

Python Streamlit Snowflake Cortex (Claude Sonnet 4.5) Live web-research API Auth0 WeasyPrint Typed JSON schemas

05 Highlights

  • Multi-agent orchestration with a typed JSON schema enforced on every agent call.
  • Dual local/AWS deploy modes supported from the same codebase.
  • Full integration test suite per agent — each agent independently testable in isolation.

06 Status
